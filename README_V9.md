# ecologue.app — patch V9 charte graphique

Ce patch reprend le V8 et ajoute l’intégration de la nouvelle identité visuelle ecologue.app.

## Ajouts principaux

- intégration des logos ecologue.app en SVG ;
- ajout d’une palette centralisée dans `config/siteConfig.ts` ;
- mise à jour de la page d’accueil `pages/HomeChoice.tsx` ;
- harmonisation du `ModeSwitcher` ;
- mise à jour du `SiteFooter` avec le logo ecologue.app et le logo Nonoké ;
- mise à jour du CTA Pro avec la nouvelle charte ;
- ajout d’un `favicon.svg` basé sur le logo icône.

## Palette intégrée

- `#F6BD60` — jaune / accent solaire
- `#F7EDE2` — fond clair principal
- `#F5CAC3` — rose pâle / fond secondaire
- `#84A59D` — vert sauge / couleur principale
- `#F28482` — corail / CTA particuliers et accent chaud

## Fichiers de marque

Les fichiers web sont copiés dans :

- `public/brand/`

Une copie source est également disponible dans :

- `assets/brand/ecologue/`

## Intégration

Remplacez les fichiers du patch V8 par ceux du V9, puis copiez les nouveaux dossiers `public/brand` et `assets/brand/ecologue` dans votre dépôt.

Consultez aussi :

- `docs/CHARTE_GRAPHIQUE_V9.md`
