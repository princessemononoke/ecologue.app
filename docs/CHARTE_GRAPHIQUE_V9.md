## Règles d’usage

- Utiliser le logo complet sur la page d’accueil, le footer et les grands écrans.
- Utiliser le logo seul pour le favicon, les boutons compacts et les contextes mobiles.
- Utiliser les versions fond foncé sur les blocs Pro sombres.
- Utiliser le corail `#F28482` pour les appels à l’action grand public.
- Utiliser le jaune `#F6BD60` comme accent chaleureux dans les blocs Pro sombres.
- Conserver le logo Nonoké dans le footer et le CTA Pro pour signer l’outil sans brouiller la marque ecologue.app.
